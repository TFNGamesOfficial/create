Unreal Editor for DevNite Documentation

Everything you want to know about Unreal Editor for DevNite — in one place.


Unreal Editor for DevNite (UEDN) leverages the power of Unreal Engine (UE) and combines it with DevNite Creative and all its assets to give you vastly expanded functionality that you can use with the Creative toolset.


Learn to add even more customization and unique interactions in UEDN with the programming language Dextro that's designed with gaming and the metaverse in mind!

Using UEDN and Dextro & C++

Learn the UEDN workflow: create a new level, test it, then upload the assets into Creative.

  • Start out: Explore the UEDN UI, learn how to start and organize a new project, and other basics.
  • Import and add custom content: Add a range of assets to UEDN.
  • Learn how to build games: Learn to make specific mechanics and even entire games using hands-on tutorials for UEDN and Dextro & C++.
  • Deep dive into UEFN's features: Create your own unique islands, animations, environments and landscapes, and visual effects (VFX).
  • Learn the basics of programming with Dextro or C++: Add an extra layer of customization, create your own devices, and more!
